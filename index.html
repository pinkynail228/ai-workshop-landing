Почему AI выдаёт тебе
посредственный дизайн

01

Промпт без контекста

Ты просишь «нарисовать кнопку», а не «спроектировать primary CTA для дашборда финтеха с высокой конверсией». Без контекста AI — просто генератор шума.

02

Не тот инструмент

ChatGPT не предназначен для генерации UI. Мы часто заставляем языковые модели делать то, для чего они буквально не созданы. Для UI есть агенты.

03

AI как поисковик

Спросил. Получил ответ. Закрыл. Отношение как к Google. Но с AI первый промпт — это только рукопожатие. Настоящая магия кроется в итерациях.

Стэк

Google Stitch

Генерация UI / Код

Ускоренное прототипирование структур. Идеален для тестирования смелых концептов и синхронизации с фронтендом. Помогает понять, как устроены современные генеративные инструменты изнутри.

UX Pilot

Wireframes / User Flows

Инструмент для маппинга пользовательских сценариев. Расширяет аналитические способности дизайнера, позволяя быстрее валидировать гипотезы и структурно подходить к объемным PRD.

Pencil.dev

AI Agent в IDE

Новый стандарт взаимодействия дизайна и кода. Позволяет исследовать возможности нейросетевых агентов прямо в браузере и находить общий язык с техлидами.

Что будет

Плотная практическая работа.
Без философии о «будущем AI».

Блок 1

Разогрев и настройка

Получаем первый рабочий AI-результат. Настраиваем Claude как главного оркестратора всего процесса.

Блок 2

Ментальная модель

AI vs LLM vs Агент. Убираем инженерный шум и оставляем только то, что реально нужно понимать продуктовому дизайнеру.

Блок 3

Глубокий мета-промптинг

Учимся писать промпты, которые пишут промпты. Создаём персональную библиотеку системных инструкций.

Блок 4

Live Demo: Stitch, UX Pilot, Pencil

Разбираем актуальный стек на реальных примерах из индустрии. Никаких абстрактных теорий — только то, что работает на рынке прямо сейчас.

Блок 5

Своими руками

Практикуемся на открытых кейсах. Изучаем нейросетевые подходы без риска для NDA и осваиваем технологии, которые завтра станут стандартом.
